# Mobile Brand Assets

Grip raster assets derived from `assets/brand/source/grip_logo_exploration_and_lockups.png` and `assets/brand/source/poe_grip_s_raven_mascot_sheet.png`.

Wired in `app.json`:

- `icon.png`
- `android-icon-foreground.png`
- `android-icon-background.png`
- `android-icon-monochrome.png`
- `favicon.png`
- `splash-icon.png`

Poe pose PNGs are transparent alpha crops for onboarding, empty states, retry/error states, and milestone/rank-up moments.
